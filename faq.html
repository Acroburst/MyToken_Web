FAQ
What is MyToken?
MyToken is an ERC-20 token, which is a decentralised cryptocurrency that is hosted on Ethereum blockchain. These tokens hold value that can be stored and sent/received using Ethereum addresses & transactions, whereby gas is used for transaction fees.
Why ERC-20?
ERC-20 is a common standard for token creation in Ethereum blockchain, and defines a set of requirements that applies to all ERC-20 tokens to allow them to interact with one another. It is also integrated and facilitated by wallets and exchanges platforms.
How do I get/use MyToken?
There is no service providers but you can request for MyToken to use for transactions from the owner and the amount can be transferred to your wallet address.
What can I find on this website?
You will find out what MyToken is about, discover the different features & functions that the ERC-20 smart contract has to offer, and interact your wallet with MyToken.
What do I do if I lost my wallet/private key?
Unfortunately, there is nothing that can be done if you have lost your private key and it is gone forever. It is extremely crucial that you backup your keys in a safe place since it is your only access to get your wallet.
Why is my device unable to load the webpage properly?
This website is a template and has not been updated to be compatible with all different types of devices yet. Ensure that you view this website on a monitor for the best user experience.
Why has my transaction not gone through?
The developers are not running the transactions, you would need to check the respective providers in question.
Why is some functions/features unavailable to me?
Make sure you have an ethereum wallet (e.g. Metamask) in your browser extension to allow the website to connect with that wallet account.
How long will it take for my question/response to be answered?
We will try our best to go through the responses regularly and get back to you as early as possible. It can take about 1 to 5 working days depending on the situation or issues at hand.
Is there any precuations I should take note of?
Ensure that the MyToken you're using for transactions is the same exact contract address as ours (0xd875bb2d9c103fcb3f69a07f4435b64c66f9d913). Since MyToken is an open source project, there's risk of impersonation of similar MyToken founders, developers and etc, thus fraud risks. You can always check the address in BscScan network.
