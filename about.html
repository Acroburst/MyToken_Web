MyToken
A BEP-20 token standard made to provide functionalities, implements an API for tokens within the smart contract and deployed to the Binance Smart Chain network. It also can be declared as ERC-20 type, since both BEP-20 and ERC-20 are compatible, in which the former is an extension of the latter. MyToken caters to the global community by representing as a virtual value or asset for your own personal needs.

The developed source code for this smart contract can be found either on the Binance Blockchain Explorer official website or through the GitHub repository below, giving you a basic understanding of how MyToken works.
